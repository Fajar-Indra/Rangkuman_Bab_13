## [cite_start]1. Isu Etika dalam TI [cite: 11]
Isu etika berfokus pada tanggung jawab sosial dan moral yang timbul dari penerapan Teknologi Informasi.

| Fokus Isu Etika | Penjelasan dan Contoh |
| :--- | :--- |
| **Privasi & Data Pribadi** | [cite_start]Melibatkan perlindungan data pribadi dan kasus kebocoran data (misal: Facebook-Cambridge Analytica [cite: 19][cite_start]). [cite: 12] |
| **Hak Kekayaan Intelektual** | [cite_start]Mencakup *software piracy* dan plagiarisme digital. [cite: 13] |
| **Transparansi Algoritma** | [cite_start]Isu *AI bias* dan *fairness* (keadilan), seperti bias gender/ras dalam penggunaan AI untuk rekrutmen. [cite: 14, 20] |
| **Tanggung Jawab Sosial** | [cite_start]Isu etika dalam Big Data & AI, termasuk misinformasi dan *deepfake*. [cite: 15, 16] |

## [cite_start]3. Keamanan dalam Strategi TI [cite: 31]

2.  **I**ntegrity (Integritas)
3.  **A**vailability (Ketersediaan)
